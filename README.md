# Comments App

A React application where users can add, like, and delete comments.

## Components
- **Comments** - Main component to add and display comments
- **CommentItem** - Individual comment with like and delete functionality

## Features
- Add comments with name and message
- Like / Unlike a comment
- Delete a comment
- Shows how long ago the comment was posted using `date-fns`

## Tech Stack
- React.js
- CSS
- date-fns
